API Endpoints

MongoDB Endpoints

  • /api/orders/active-products - Get active product IDs
  • /api/orders/by-status?status=COMPLETED - Get orders by status
  • /api/db-stats/collections - Get database statistics

Elasticsearch Endpoints

  • /api/elasticsearch/products/active - Get active products via Elasticsearch
  • /api/elasticsearch/products/active/optimized - Get active products via optimized Elasticsearch

Redis Cached Endpoints

  • /api/redis-cached/products/active - Get active products with Redis caching

Dual-Write Architecture Endpoints

  • /api/products/active - Get active product IDs (Redis with MongoDB fallback)
  • /api/products/consistency-check - Check consistency between MongoDB and Redis

Order Management for Active Products

  • /api/orders/active - Get all orders for active products
  • /api/orders/active-products - Get active products with orders

Performance Testing

  • /api/performance/test?iterations=3&concurrent=5 - Run performance tests
  • /api/performance/warmup - Warm up systems (POST)
